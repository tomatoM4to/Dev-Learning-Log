Objects

  • JS에서는 원시 타입(primitive type)인 숫자, 문자열, 불리언, null, undefined, 심볼 외에도 객체 타입이 있다.
  • 데이터, 함수들의 묶음이다.
  • JS의 모든 객체는 최상위 객체인 Object의 인스턴스이다.
